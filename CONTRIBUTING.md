# Contributing

We welcome issue reports and tool suggestions.

The **block-hooks** tool lives under [`hooks/block-hooks/`](hooks/block-hooks/): `install.sh`, `block-hooks.sh`, `keywords.txt`, and `keywords.example.txt`. Clone the repo, `cd hooks/block-hooks`, and run `bash install.sh`.

Optional regression checks from the repository root: `bash tests/run.sh`.

For now, pull requests are reviewed on a best-effort basis by a small team. If you have built something that addresses a real AI agent risk, open an issue first describing the tool and the threat it addresses. We will let you know quickly if it is a fit.

By contributing you agree your code will be licensed under Apache 2.0.
